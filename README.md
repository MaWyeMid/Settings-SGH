### 简介
从易格尔兰殖民地到世界第一超级大国，短短300多年间，格内茨共和国或许得到了上帝的眷顾，在历史分岔路上的每一步都踩在了正确的分支上，吗？抛开今天格内茨共和国存在的种种问题不谈，今天我们将用通俗的语言，给你讲述格内茨共和国从殖民地时期，一直到现在的历史，或是新闻。本书会持续连载，完结时间待定。享受像读小说一般学习历史吧！
